# 🎮 Wordle Clone

A modern **Wordle Clone** built with **React** and **Vite**, featuring a responsive design, smooth animations, and the classic Wordle gameplay experience.

## ✨ Features

- 🎯 Classic 5-letter Wordle gameplay
- 🎨 Modern dark UI
- ✨ Smooth tile flip and typing animations
- 📱 Fully responsive for desktop and mobile
- ⌨️ Physical & on-screen keyboard support
- 📊 Guess counter
- 🏆 Game over modal
- 🎲 Random word generation

## 🚀 Tech Stack

- React
- Vite
- CSS3

## 📦 Installation

```bash
git clone https://github.com/Rehman481/wordle-game.git
cd wordle-game
npm install
npm run dev
```

Open **http://localhost:5173** in your browser.

## 🎮 How to Play

- Guess the hidden **5-letter word** in **6 attempts**.
- Press **Enter** to submit a guess.
- Press **Backspace** to delete a letter.

### Tile Colors

- 🟩 **Green** — Correct letter in the correct position.
- 🟨 **Yellow** — Correct letter in the wrong position.
- ⬜ **Gray** — Letter is not in the word.
